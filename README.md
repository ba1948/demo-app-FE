## Installattion 
run `npm install` and make sure the file 'proxy.conf.json' is existing and points to the correct backend.

## Development server

Run `npm run start` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

Make sure that the BE for this demo app is install and running, you can find instructions at: https://github.com/ba1948/demo-app-BE
